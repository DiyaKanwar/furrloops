When updating the FurrLoops project

- Always check if the README.md in trickle/notes needs updating
- Update README.md when:
  - New pages or features are added
  - Design system changes (colors, fonts, styles)
  - Project structure modifications
  - New product categories added
  - Important functionality changes
- Keep README.md concise and well-organized
- Maintain sections: Overview, Design System, Website Structure, Technical Stack, Brand Values
